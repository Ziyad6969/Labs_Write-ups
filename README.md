# Labs Write-ups

This repository contains concise write-ups for security labs and challenges
solved across multiple platforms.

The focus of these write-ups is on:
- Understanding the root cause of vulnerabilities
- Exploitation logic and key takeaways
- Recognizing recurring patterns across platforms

## Platforms
- PortSwigger Web Security Academy
- CyberTalents

## Structure
Write-ups are organized by platform, then by vulnerability category or challenge type.
