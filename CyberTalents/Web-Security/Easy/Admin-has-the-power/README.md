# Admin Has the Power

**Platform:** CyberTalents  
**Difficulty:** Easy
**Category:** Authentication / Access Control
**[Lab](https://cybertalents.com/challenges/web/admin-has-the-power)**

<div>

## Objective

Only administrators are allowed to view the flag.  
Your goal is to obtain admin privileges and retrieve the flag.

---

## Lab Overview

Upon opening the challenge, you are presented with a login page that requires a username and password to proceed.

No additional functionality is available until authentication is completed.
