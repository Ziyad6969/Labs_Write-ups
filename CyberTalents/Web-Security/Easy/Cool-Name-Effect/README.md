# Cool Name Effect

**Platform:** CyberTalents  
**Difficulty:** Easy  
**Category:** Web Security  

---

## Lab

### Objective

Webmaster developed a simple script to do cool effects on your name, but his code is not filtering the inputs correctly.

Execute a JavaScript alert and prove it.

---

## Lab Overview

Upon opening the lab, we are presented with a form where we can enter our name.

After submitting the form, the entered name is displayed in a curved arch-like format on the page.

The goal of the challenge is to perform Cross-Site Scripting (XSS) and execute JavaScript code.
